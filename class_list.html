Tailwind CSS Class List

Category Class Description
Layout & Container container Centers and applies padding based on the screen size
block Display block
inline-block Display inline-block
inline Display inline
flex Display flex
inline-flex Display inline-flex
grid Display grid
hidden Hides element (display: none)
overflow-hidden/auto/scroll Controls content overflow (hidden, auto, or scroll)
object-contain/cover Object-fit control for images
Spacing (Padding & Margin) p-{size} Padding (e.g., p-4, p-0, p-x-4)
pt-{size}, pb-{size} Padding Top / Bottom
pl-{size}, pr-{size} Padding Left / Right
m-{size} Margin (e.g., m-4, m-0, m-auto)
mt-{size}, mb-{size} Margin Top / Bottom
ml-{size}, mr-{size} Margin Left / Right
Typography text-xs to text-6xl Text size
font-thin to font-black Font weight from thin to black
italic / not-italic Italic or not
underline / no-underline Text decoration
uppercase / lowercase / capitalize Text transformation
tracking-{size} Letter spacing (e.g., tracking-wider)
leading-{size} Line-height control (e.g., leading-loose)
text-left/right/center Text alignment
truncate Truncate text overflow
